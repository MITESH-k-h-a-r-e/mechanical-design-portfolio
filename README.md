# SolidWorks CSWA Models

A collection of **SolidWorks CAD models, assemblies, and drawings** created during preparation for the **Certified SOLIDWORKS Associate (CSWA)** certification. This repository showcases fundamental mechanical design skills, parametric modeling techniques, and engineering drawing practices.

---

## Overview

This repository includes CAD files covering the core topics tested in the CSWA examination, including:

* 3D Part Modeling
* Parametric Feature Design
* Assemblies
* Engineering Drawings
* Configurations
* Material Assignment
* Mass Property Validation
* Design Intent

---

## Skills Demonstrated

* Parametric CAD Modeling
* Sketch Relations & Fully Defined Sketches
* Extrude, Revolve, Sweep & Loft Features
* Fillets & Chamfers
* Linear, Circular & Mirror Patterns
* Shell & Draft Features
* Reference Geometry
* Assembly Mates
* Engineering Drawings (GD&T excluded)
* Material Assignment
* Mass Properties
* Design Intent & Model Editing

---

## Software

* **CAD Software:** SOLIDWORKS
* **Certification Level:** CSWA
* **File Formats:**
